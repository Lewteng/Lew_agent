# Lew_agent 🎬🧠🔊

一个多模态智能体项目，实现 **图片理解 → 故事生成 → 语音播报 → 多轮对话** 的完整链路。

> 从一个图片开始，让 AI 讲故事，并用声音“说出来”。

---

## ✨ 项目功能

### 🖼️ 1. 图片生成故事
- 输入图片 + 提示词
- 基于视觉大模型生成完整故事文本
- 支持中文自然语言表达

### 🔊 2. 故事转语音
- 支持多种语音风格：
  - 默认女声
  - 温和男声
  - 讲故事男声（推荐🔥）
  - 快速语音
- 自动生成并播放音频

### 💬 3. 多轮智能对话
- 支持上下文记忆
- 可结合“最近生成的故事”继续交流
- 统一为 Lew_agent 助手身份

### 🔗 4. 多模态联动（核心亮点）
- 图片 → 故事 → 语音 → 对话
- 聊天中可继续加工刚才的故事

---

## 🧠 技术架构

```text
用户输入（图片 / 文本）
        ↓
视觉模型（QVQ）
        ↓
故事生成（LLM）
        ↓
TTS语音合成（edge_tts）
        ↓
音频输出
        ↓
多轮对话（ChatAgent）
